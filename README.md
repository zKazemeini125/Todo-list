# Todo List Application

A simple and beginner-friendly Todo List project designed for those who are just starting their journey in web development.

## 🚀 About the Project
This project was originally built using pure **HTML, CSS, and JavaScript**. It serves as a perfect example of managing state and DOM manipulation without any external frameworks.

### 🛠️ Refactored Version
This repository now contains a **refactored version** of the project. The code has been restructured for better readability, maintainability, and efficiency.
* **Entry Point:** The project execution begins in the `index.js` file.

## ✨ Features
- **CRUD Operations:** Easily create, edit, and delete tasks.
- **Data Persistence:** Tasks are automatically saved in `localStorage`.
- **Search & Highlight:** Search through your tasks with real-time highlighting of results.
- **Sorting:** Organize your tasks efficiently.
- **UI/UX Components:** 
  - Smooth Modal windows for task management.
  - Toast notifications for intuitive user alerts.

## 💻 Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
